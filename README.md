## üìù Lab Overview

### **1. Wireless Settings**
- Wireless Mode: **802.11 b/g/n mixed**
- SSID Name: **Home Wi-Fi**
- Channel Selection: **Auto**
- Router Interface: **TRENDnet**

These settings ensure compatibility with a wide range of devices while allowing the router to automatically select the optimal channel.

### **2. Security Configuration**
- SSID Selection: **Home Wi-Fi**
- Security Mode: **WPA2-PSK**
- Encryption Algorithm: **AES**
- Passphrase: **H@rd2Guess**

WPA2-PSK with AES is the recommended standard for securing home wireless networks.
